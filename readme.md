# Docker Assignment 1

##  Description

This project demonstrates containerizing a Python Flask application using Docker.

##  Technologies Used

* Python
* Flask
* Docker

##  Steps to Run

### 1. Build Docker Image

docker build -t python-app .

### 2. Run Container

docker run -d -p 5000:5000 python-app

### 3. Open in Browser

http://localhost:5000
